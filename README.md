# WorkBuddy AI 对话记忆存档

## 仓库说明

本仓库用于永久存档 WorkBuddy AI 助手的对话记忆，确保跨会话、跨时间的长期记忆。

## 目录结构

```
/
├── memories/           # 按年份组织的对话记忆
│   ├── 2026/
│   │   ├── 2026-04.md  # 2026年4月对话摘要
│   │   └── ...
│   └── ...
└── projects/           # 项目相关的长期记忆
    └── ...
```

## 使用说明

每次对话结束后，相关摘要会自动整理并提交到本仓库。
